# Simple Task Tracker Web App

A simple Task Tracker web app built with  React .This app allows users to add, mark and delete tasks efficiently.




## Features

- Add new tasks .
- Mark tasks as completed.
- Delete tasks from the list.

## Technologies Used

-Built fully in **React**


## Usage

Once the application is running, you can:

1. Add a new task by typing in the input field and clicking the "Add" button.
2. Mark tasks as completed by checking the checkbox next to each task. it.
3. Delete a task by clicking the 'delete' button next to it.
